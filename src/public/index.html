About ShieldVault

🛡️ ShieldVault v1.0

A production-grade secure file upload and virus scanning service. Every file is encrypted with AES-256-CBC before storage. ClamAV antivirus scans every upload and quarantines infected files automatically.

Framework
Node.js + Express
Encryption
AES-256-CBC
Virus Scanner
ClamAV (via Docker)
Auth
JWT (24h expiry)
Database
SQLite (local)
Download Links
HMAC-SHA256 Signed URLs (1h)
Max File Size
10 MB
Allowed Types
Images, PDF, TXT, CSV, ZIP, DOCX
⚠ Demo Mode: If ClamAV is not running, the scanner operates in demo mode — files are still encrypted and stored, but virus scanning is simulated. Run with Docker Compose to enable real ClamAV scanning.
